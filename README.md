## 📖 О проекте

**Oceania VPN** — это современный VPN-сервис, созданный для обеспечения вашей конфиденциальности, безопасности и свободы в сети. Мы используем передовые протоколы шифрования и технологию обфускации трафика, чтобы гарантировать стабильное соединение и обход любых сетевых ограничений.

## ✨ Ключевые возможности

* 🛡️ **Полная конфиденциальность (Zero-Log)** — мы не собираем, не храним и не передаем логи вашей активности.
* ⚡ **Максимальная скорость** — инфраструктура на базе WireGuard обеспечивает минимальные задержки и высокую пропускную способность.
* 🧅 **Обход блокировок** — встроенная обфускация (Reality / VLESS / Shadowsocks) маскирует VPN-трафик под обычный HTTPS.
* 🌍 **Глобальная сеть** — серверы в 15+ странах с автоматическим выбором лучшего узла.
* 📱 **Мультиплатформенность** — нативные клиенты для всех популярных операционных систем.

### 2. Авторизация и подключение
1. Откройте приложение и войдите в свой аккаунт.
2. Выберите режим **«Быстрое подключение»** (Smart Connect) или укажите нужную страну вручную.
3. Нажмите кнопку питания. Готово!

### ⌨️ Для терминала (Linux / CLI)
```bash
# Установка CLI-клиента
curl -fsSL https://get.oceania-vpn.com/install.sh | bash

# Подключение к лучшему серверу
oceania connect --auto

# Подключение к конкретному региону через Reality
oceania connect --region de --protocol reality
```
